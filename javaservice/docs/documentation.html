Document Sets

The JavaService documentation is included as part of the software distribution, and is also available for reference on the Internet.

Documentation is split into different areas of interest, with some common documents being referenced from both areas.
These sections are briefly described below and are accessed using the documentation menu above, or from the links below.

Current Documentation

 •  User Documentation

This documentation is provided to explain how the software is seen by the end user.
It lists the commands available from the utility for installing, querying and removing a Java application as a service within the Windows environment.

 •  Developer Documentation

This documentation is intended for technical users who may be developing a Java application to be run as a service.
It specifies the lower-level mechanisms used by the software and defines the application interface that is uses.

Previous Documentation

 •  Version 1.2 Documents

This documentation is included for reference only, being a copy of the earlier document set provided with earler (pre-V2) releases of JavaService.

Internet Information

 •  ObjectWeb Project Page

The JavaService Project Homepage is hosted as part of the ObjectWeb Consortium open source initiative. This site has copies of the documentation set for JavaService, plus additional information and links to the source forge.

 •  ObjectWeb Source Forge

The JavaService Source Forge on ObjectWeb provides access to the source file repository, software downloads, news items, mailing lists and support forums.
